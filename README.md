## CampusNest

CampusNest is a mobile application designed to help students easily find and explore available hostels near their campus. The app provides information such as location, pricing, and availability to simplify the hostel search process.

## Features

 Search for hostels within a specific distance
 View hostel locations
 Check pricing and availability
 Browse different hostel options
 User-friendly interface for students

## Technologies Used

 Kotlin (Android Development)
 Firebase (Database & Backend Services)
 XML (UI Design)

 ## Getting Started

 **Prerequisites**

* Android Studio installed
* Basic knowledge of Kotlin
* Firebase account

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/IAM-NO-4/campusNest1.git
   ```
2. Open the project in Android Studio
3. Sync Gradle and install dependencies
4. Connect the app to Firebase
5. Run the app on an emulator or physical device

##  Project Structure

* `app/` - Main application code
* `res/` - UI resources (layouts, drawables, etc.)
* `java/` or `kotlin/` - Source code files
